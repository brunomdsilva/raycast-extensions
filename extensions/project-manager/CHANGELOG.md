# Project Manager Changelog

## [Initial Version] - 2025-01-29

- Manage project paths with wildcard support (`~/Developer/*` for subfolders, `~/Developer/my-app` for single projects)
- List, search and open projects in editor, terminal, or both
- Create and delete projects
- Pin projects to the top of the list
- Sort projects by frequency, manual order, alphabetical, or recent
- Path validation with "Not found" indicator for invalid paths
- Interactive path management with search-as-add pattern
- Help page documenting path formats
