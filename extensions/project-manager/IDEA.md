# Quick Code

Extensão do Raycast para organização e acesso rápido a projetos de desenvolvimento.

## Conceito

O usuário configura os caminhos dos seus projetos e a extensão lista todos eles para acesso rápido.

## Configuração do usuário

- **Caminhos de projetos**: lista de caminhos com suporte a wildcard
    - `~/Developer/*` — todas as pastas dentro do diretório
    - `~/Developer/more-projects/*` — outros caminhos
    - `~/Developer/some-path/specific-project` — projeto único (caminho específico)
- **Editor principal**: aplicativo padrão para abrir projetos
- **Terminal principal**: aplicativo padrão para abrir terminal

## Funcionalidades

- Listar todos os projetos configurados
- Pesquisar projetos por nome
- Abrir projeto no editor principal (ação padrão)
- Abrir projeto no terminal
- Abrir projeto no editor + terminal
- Abrir projeto com "Abrir com..." (escolher aplicativo)
- Criar novo projeto (escolher pasta destino)
- Deletar projeto
- Mover projeto
- Limpar histórico de pesquisas
- Limpar histórico de projetos abertos
